# 🛍️ Toko Prodak Onlen


Toko Prodak Onlen adalah website marketplace produk digital yang menyediakan berbagai produk seperti UI Kit, E-book, Preset Foto, Font, Plugin, dan Kursus. Website ini memiliki fitur pencarian produk, kategori, promo, serta panduan pembelian dengan tampilan yang responsif menggunakan HTML dan CSS.

## Fitur
- Landing page modern
- Navbar dengan menu navigasi
- Kolom pencarian produk
- Hero section
- Kategori produk
- Promo produk
- Langkah pembelian
- Footer dengan informasi kontak
- Responsive untuk Desktop, Tablet, dan Mobile

## Teknologi
- HTML5
- CSS3
- Font Awesome
- Google Fonts (Poppins)

## Struktur Folder
```
project/
│── index.html
│── style.css
│── assets/
│   ├── images/
│   └── icons/
└── README.md
```

## Cara Menjalankan
1. Download atau clone repository.
2. Buka folder proyek.
3. Jalankan `index.html` menggunakan browser atau Live Server di Visual Studio Code.

## Tujuan
Website ini dibuat sebagai proyek pembelajaran Frontend Web Development dengan fokus pada pembuatan tampilan marketplace produk digital yang sederhana, responsif, dan mudah digunakan.
